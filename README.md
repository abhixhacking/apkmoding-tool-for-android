# APK Modding Tool for Android

> A complete toolkit for modding APKs directly from your Android phone – no PC required.

This repository contains all the essential apps you need to edit, repack, sign, and test modified APKs right on your mobile device.

## 📦 Included Tools

| File | Purpose |
|------|---------|
| `MT_Manager_premium.apk` | Powerful file manager with built-in APK editor, decompiler, and signer (premium version) |
| `Developer Assistant_1.2.2.apk` | Helps with app debugging, viewing activities, and testing mods |
| `Shizuku_13.5.4.r1049...apk` | Grants elevated permissions to apps without root |
| `LSPosed_1.9.2.apk` | Xposed Framework replacement for system-level hooking (requires root / Riru) |
| `Dual Space_1.13.0.11.apk` | Create parallel/dual apps – useful for testing modded APKs alongside originals |
| `ZArchiver_1.0.10.apk` | Extract and compress ZIP, RAR, 7z files – necessary for unpacking APK resources |

## 🚀 How to Use

1. **Install all APKs** on your Android device.
2. **Open MT Manager** – it will be your main modding tool.
3. Use **ZArchiver** to extract any APK or resource files.
4. Edit files (XML, smali, etc.) inside MT Manager.
5. Repack and sign the modified APK using MT Manager's built-in signer.
6. (Optional) Use **Shizuku** for deeper access without root, or **LSPosed** for hooking if rooted.

## ⚠️ Requirements

- Android 8.0 or higher recommended
- Some tools may require root access (LSPosed)
- Enable "Install from unknown sources" in settings
